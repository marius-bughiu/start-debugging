Lighthouse es una herramienta automatizada para auditar el rendimiento, la accesibilidad y la experiencia de usuario general de cualquier página web. Inicialmente estaba pensado para auditar PWAs (Progressive Web Apps), pero es mucho más que eso.

La forma más sencilla de auditar tu sitio web es usar la última versión de [Chrome](https://www.google.com/chrome/b/). Navega a tu sitio y abre las Dev Tools pulsando F12 o haciendo clic derecho en la página > Inspect (Ctrl + Shift + I). A continuación ve a la pestaña Audits y haz clic en Run audits. Esto ejecutará un conjunto de pruebas contra tu sitio y te dará un informe como el siguiente:

![Google Lighthouse - resumen del informe](/wp-content/uploads/2019/04/image.png)

Como puedes ver, este blog no anda muy bien. Por suerte, Lighthouse te da un informe muy detallado que explica tu puntuación y cada línea va acompañada de un enlace Learn more donde puedes conocer más detalles sobre el problema y cómo mejorar tu valoración.

![Google Lighthouse - informe de Opportunities](/wp-content/uploads/2019/04/image-1.png)

Puedes jugar con las opciones, probar con un dispositivo móvil o usar diferentes velocidades de conexión que coincidan con tu público objetivo y generar distintos informes. Una vez que tengas tu línea base, llega el momento de empezar a mejorar esos problemas.
