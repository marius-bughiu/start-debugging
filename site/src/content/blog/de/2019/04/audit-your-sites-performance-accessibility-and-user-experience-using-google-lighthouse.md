Lighthouse ist ein automatisiertes Tool zum Auditieren der Performance, Barrierefreiheit und allgemeinen Nutzererfahrung beliebiger Webseiten. Ursprünglich war es für das Auditieren von PWAs (Progressive Web Apps) gedacht, ist mittlerweile aber viel mehr als das.

Am einfachsten auditieren Sie Ihre Website mit der neuesten Version von [Chrome](https://www.google.com/chrome/b/). Navigieren Sie zu Ihrer Website und öffnen Sie die Dev Tools mit F12 oder per Rechtsklick auf die Seite > Inspect (Ctrl + Shift + I). Wechseln Sie dann zum Tab Audits und klicken Sie auf Run audits. Damit wird eine Reihe von Tests gegen Ihre Website ausgeführt, und Sie erhalten einen Bericht wie diesen:

![Google Lighthouse - Berichtsübersicht](/wp-content/uploads/2019/04/image.png)

Wie Sie sehen, schneidet dieser Blog nicht besonders gut ab. Glücklicherweise liefert Lighthouse einen sehr detaillierten Bericht, der Ihren Score erklärt, und jede Zeile wird von einem Learn-more-Link begleitet, über den Sie weitere Details zum Problem und Wege zur Verbesserung Ihrer Bewertung erfahren.

![Google Lighthouse - Opportunities-Bericht](/wp-content/uploads/2019/04/image-1.png)

Sie können mit den Einstellungen experimentieren, mit einem mobilen Gerät testen oder verschiedene Verbindungsgeschwindigkeiten passend zur Zielgruppe simulieren und unterschiedliche Berichte erzeugen. Sobald Sie Ihre Baseline haben, ist es Zeit, an diesen Punkten zu arbeiten.
