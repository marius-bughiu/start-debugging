Сегодня, пытаясь (по ошибке) обновить свою Windows 7 до Windows 8, я наткнулся на проверку совместимости с 6 ошибками, среди которых:

> Secure Boot isn't compatible with your PC

Сначала меня обдало холодком от мысли, что не получится установить Windows 8, хотя Consumer Preview работала нормально (с парой исключений), но пара поисков спустя я понял, что это вовсе не проблема. Secure Boot - это просто функция, которую можно пропустить, и всё будет работать как надо.

**Так что же такое Secure Boot?**

Secure Boot - это новый процесс загрузки (measured boot), который вместе с UEFI 2.3.1 (Unified Extensible Firmware Interface) устраняет существующую брешь в безопасности нынешнего дизайна BIOS, позволяющую загружать вредоносное ПО до операционной системы. Это делается с помощью сертификатов - по сути, ничего не будет загружено, если оно не подписано Microsoft, то есть никакого malware.

Эта функция доступна только на относительно новых системах, потому что опирается на чип Trusted Platform Module (TPM). Этот чип используется для хранения подписанных, защищённых и измеренных процессов запуска, на которых строится Secure Boot.

Так что нет TPM - нет Secure Boot, только обычная загрузка. То есть это не помешает вам установить Windows 8 на вашу машину.
