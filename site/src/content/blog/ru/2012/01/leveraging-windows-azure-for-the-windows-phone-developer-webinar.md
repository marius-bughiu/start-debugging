Сообщаю, что 4-го числа этого месяца будет вебинар по Windows Azure и тому, как нам им воспользоваться в Windows Phone. Вебинар проведёт Samidip Basu, повестка такая:

-   Introduction to Cloud Computing & Windows Azure
-   Push Notifications for Windows Phone
-   Push Notifications from Azure, including deep-linked secondary Live Tiles & Toasts for Windows Phone Mango
-   Why consider OData while leveraging the cloud in cross-platform Mobile solutions
-   Usage of SQL Azure & OData through Windows Phone
-   Usage of Azure-hosted WCF Services for Windows Phone
-   How to use Azure Toolkit for Windows Phone

Дата и время: 4 января, 10 AM PDT
Ссылка на регистрацию: больше недоступна

Написав о вебинаре в Twitter с тегом **#webinarsilverlightshow**, вы получаете шанс выиграть [Windows Phone 7 Silverlight Cookbook](http://www.silverlightshow.net/items/Book-Review-Window-Phone-7-Silverlight-Cookbook.aspx); а в конце вебинара, оставив конструктивный отзыв, можно бесплатно получить ebook [OData & Cloud Augmentation of Windows Phone Applications](http://www.silverlightshow.net/ebooks/odata_cloud.aspx).

Подробнее о вебинаре и бесплатных ebook см. на [SilverlightShow.NET](http://www.silverlightshow.net/news/Free-SilverlightShow-Webinar-Leveraging-Windows-Azure-for-the-Windows-Phone-Developer.aspx "SilverlightShow.NET").
