今月 4 日に、Windows Azure と、それを Windows Phone でどう活用できるかについてのウェビナーがあるのでお知らせします。プレゼンターは Samidip Basu で、アジェンダは次のとおりです。

-   Introduction to Cloud Computing & Windows Azure
-   Push Notifications for Windows Phone
-   Push Notifications from Azure, including deep-linked secondary Live Tiles & Toasts for Windows Phone Mango
-   Why consider OData while leveraging the cloud in cross-platform Mobile solutions
-   Usage of SQL Azure & OData through Windows Phone
-   Usage of Azure-hosted WCF Services for Windows Phone
-   How to use Azure Toolkit for Windows Phone

日時: 1 月 4 日 10 AM PDT
登録リンク: 現在は利用できません

ハッシュタグ **#webinarsilverlightshow** を付けてこのウェビナーをツイートすると、[Windows Phone 7 Silverlight Cookbook](http://www.silverlightshow.net/items/Book-Review-Window-Phone-7-Silverlight-Cookbook.aspx) が当たるチャンスがあります。また、ウェビナーの最後に建設的なフィードバックを残せば、ebook [OData & Cloud Augmentation of Windows Phone Applications](http://www.silverlightshow.net/ebooks/odata_cloud.aspx) を無料で手に入れられます。

ウェビナーや無料 ebook の詳細は [SilverlightShow.NET](http://www.silverlightshow.net/news/Free-SilverlightShow-Webinar-Leveraging-Windows-Azure-for-the-Windows-Phone-Developer.aspx "SilverlightShow.NET") をご覧ください。
